# AI Assistant Core API (Project 5)

This project is a simple AI Assistant built using Python that can interact with real-world APIs to provide useful information such as weather updates, currency conversion, and latest news.

##  Features

-  Weather lookup by city
-  Currency conversion using live exchange rates
-  Latest news headlines
- Simple rule-based AI assistant logic

##  Tech Stack

- Python
- Requests library
- Public APIs (weather, currency, news)

##  Project Structure

assistant.py → Main AI brain that handles user input  
weather.py → Fetches weather data  
currency.py → Handles currency conversion  
news.py → Fetches latest news  
utils.py → Helper functions
